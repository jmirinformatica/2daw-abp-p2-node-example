# Exemple de nodejs i react amb Server-Sent Events (SEE) 

Codi fet a partir d'aquest article:

* https://medium.com/@imanshurathore/server-sent-events-in-react-30021f9ffc4a

En primer lloc, cal iniciar el [server](./server/):

    cd server
    npm install
    npm run dev

Si es vol fer servir el [client](./client/):

    cd client
    npm install
    npm run dev
